# gptFun小助理
使用GPT API最新的function call实现的智能API调度功能，可以帮助你搜索音乐、查找天气、搜索微博热搜以及在GitHub上搜索代码。

体验传送门：https://aiquickhelp.com/gptFun/

## 音乐搜索和播放，基于私搭的网易云API（接口就不公开了，大伙感兴趣可以搜索相关的github项目去搭建）
